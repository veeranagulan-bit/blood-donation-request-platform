# 🩸 Blood Donation Request Platform

A simple web application that helps users register as blood donors, search for donors by blood group, view donor locations on a map, and check blood donation eligibility.

## ✨ Features

- Donor Registration
- Search Donors
- Interactive Map
- Eligibility Checker
- Blood Donation Chatbot
- Responsive Interface

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MongoDB
- Leaflet.js
- OpenStreetMap

## 📂 Project Structure

```
Blood-Donation-Platform/
│── index.html
│── register.html
│── style.css
│── script.js
│── server.js
│── package.json
└── README.md
```

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/Blood-Donation-Platform.git
```

2. Install dependencies

```bash
npm install
```

3. Start MongoDB.

4. Run the server

```bash
node server.js
```

5. Open in your browser:

```
http://localhost:3000
```
